Partiful Profiles

Introduction

We started the process with 32 rough sketches; ideating on 8 different 4 screen flows.

❮ ❯

Wireframes

From here, we compiled the sketches into some initial wireframes. Our primary objective was incentivizing users to complete their profile with a photo and more information. We focused on two avenues to achieve this:

  1. Adding more calls to action and paths to direct users towards editing their profile
  2. Emphasizing the social connectivity of the platform with the assumption that if you feel like people are actually looking at your profile, you are more likely to care about how it looks.

For our first objective we added more customizable features since we believe that if people want to edit or fill out one feature on their profile they are more likely to fill out more things since they are already on their profile. Some new features we designed to achieve this included:

  1. "Party anthem" module on the profile page. A fun feature that can help bring people to the edit profile page
  2. Rebranding birthday month as horoscope on profile page.This feels less revealing of personal info and is more interesting to know
  3. Adding complete check mark on profile that can unlock new customization features
  4. Including buttons prompting the user to add socials by default when viewing the user's own profile
  5. Add new customization options for profile background (similar to event page)

High Fidelity Mockups

❮ ❯

Hi-Fi Design choices

Our high fidelity mockups include the following features, aimed at incentivizing users to complete their Paritful profiles:

Complete check marks

Everyone wants the verified blue checkmark on social media platforms like Instagram or Twitter, so we decided to leverage this motivation to incentivize users to complete their profiles. Once a profile is completely filled out, users get a yellow checkmark icon that appears next to their name.

Gatekeeping socials

Users can't see other user's linked socials unless they have added socials themselves. This incentivizes users to avoid missed connections from parties by adding socials so partiers can get in touch.

Edit profile flow

We added a progress bar to show how complete a user's profile is. This can help inform the user that there still things for them to complete, and to incentive users's to fill up the bar..

Party anthems

We added the option to link a song from Spotify or Apple Music to a profile so that users can share more of their personality! This is a field that we think users will want to complete, which adds more information to their profile page.

Pop-ups

We believe that adding more prompts with compelling reasons for users to add information will increase the completion of profiles. Adding fun pop-ups can remind the user that they haven't completed their profile while still allowing for easy dismissal in order not to be a burden. Our pop-ups also provide a reason for why Partiful is asking for this information.

Customizable background

Once a user completes all the information from their profile, they unlock customizable backgrounds and particles effects, similar to the events page. The idea is that users will enjoy the fun customization, and therefore be incentivized to complete their profiles.

Cards

We modified the guest list of an event from a scrollable section of names and profile pictures, to featuring "cards" for each profile. These cards provide users with a fun way to showcase their customized profile settings even when in a list among other users. By expanding the effects of customization beyond a user's profile page, users are further incentivized to complete their profile.
